## 📥 HOW TO UPLOAD

### Step 1: Download Repository
```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### Step 2: Create Shopify ZIP

**Include ONLY these folders:**
- `layout/`
- `sections/`
- `templates/`
- `config/`
- `snippets/`
- `locales/`
- `assets/` (only .css and .js files: `averra-theme.css`, `base.css`, `global.js`)

**Mac/Linux:**
```bash
zip -r averra-shopify-complete.zip \
  layout/ \
  sections/ \
  templates/ \
  config/ \
  snippets/ \
  locales/ \
  assets/averra-theme.css \
  assets/base.css \
  assets/global.js
```

**Windows:**
1. Create folder `averra-shopify-complete`
2. Copy the 7 folders above into it
3. Right-click → Compress to ZIP

### Step 3: Upload to Shopify
1. Login: `https://your-store.myshopify.com/admin`
2. Go to: **Online Store** → **Themes**
3. Click: **Add theme** → **Upload ZIP file**
4. Select: `averra-shopify-complete.zip`
5. Wait for upload (1-2 minutes)
6. Click: **Customize**
7. Click: **Publish**
