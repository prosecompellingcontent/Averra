# 🏆 AVERRA AI Model Studio - Complete Shopify Theme

## ✅ FULLY CONVERTED TO SHOPIFY

This repository contains a **complete, production-ready Shopify theme** for AVERRA AI Model Studio. The entire site has been converted from React to native Shopify Liquid with NO visual changes.

## 🚀 Quick Start

### 1. Download This Repository
Click the green **"Code"** button → **"Download ZIP"**

### 2. Create Shopify-Ready ZIP

**Include ONLY these 7 folders:**
- `layout/`
- `sections/`
- `templates/`
- `config/`
- `snippets/`
- `locales/`
- `assets/` (only .css and .js files)

### 3. Upload to Shopify
1. Go to: **Online Store** → **Themes**
2. Click: **Add theme** → **Upload ZIP file**
3. Upload your ZIP
4. Click: **Customize** → **Publish**

**📖 Detailed Instructions:** See `/COMPLETE_CONVERSION_README.md`

## 🆘 Troubleshooting

### "Missing layout/theme.liquid" Error
- Make sure `/layout/theme.liquid` exists in your ZIP
- Check that folders are at ZIP root level (not nested)

### Sections Not Showing
- After upload, go to Customize
- Click "Add section"
- Select from dropdown

### Styles Not Loading
- Verify `/assets/averra-theme.css` is included
- Check browser console for errors

### Cart Not Working
- Uses Shopify's native cart
- Ensure products are created in Admin
