## 🚀 Upload to Shopify

### Mac/Linux - Create ZIP:
```bash
cd your-repository-folder

zip -r averra-shopify-final.zip \
  layout/ \
  sections/ \
  templates/ \
  config/ \
  snippets/ \
  locales/ \
  assets/averra-theme.css \
  assets/base.css \
  assets/global.js
```

### Windows - Create ZIP:
1. Create new folder called `averra-shopify-final`
2. Copy these 7 folders INTO it:
   - `layout/`
   - `sections/`
   - `templates/`
   - `config/`
   - `snippets/`
   - `locales/`
   - `assets/` (only copy the 3 .css and .js files)
3. Right-click the folder
4. Select "Send to" → "Compressed (zipped) folder"

### Upload:
1. Login to Shopify: `https://your-store.myshopify.com/admin`
2. Go to: **Online Store** → **Themes**
3. Click: **Add theme** → **Upload ZIP file**
4. Select: `averra-shopify-final.zip`
5. Wait 1-2 minutes for upload
6. Click: **Customize** button
7. Add content and images
8. Click: **Publish**
