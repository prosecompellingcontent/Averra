Immediate Stabilization Plan (Do This Now)
1️⃣ Create a Clean Shipping File

On desktop:

Duplicate the file.

Rename it: AVERRA_MARCH3_SHIP

Delete:

All old versions

All draft pages

All mood boards

All unused components

Any archive sections

Hidden pages still load into memory. Delete them.

2️⃣ Replace Heavy Images (This Is Usually 80% of the Problem)

If you used AI-generated images, they’re likely:

4000–8000px wide

PNG

Massive file size

Re-export them:

Hero images → 2000–2500px wide max

Section images → 1200–1600px wide

JPG quality 70–80%

Use PNG only if transparency is required

Then replace them in Figma.

Do not leave original full-res files inside.

3️⃣ Remove Real-Time Blur

Mobile browsers hate:

Background blur

Layer blur

Multiple transparency overlays

If you used blur for luxury effect:

Export the blurred background once as an image and re-import it.
Do not use live blur effects.

4️⃣ Flatten Decorative Layers

Select complex hero sections:
Right click → Flatten

Keep:

Text

Buttons

Interactive elements

Flatten:

Image compositions

Overlays

Decorative framing

5️⃣ Remove Excess Smart Animate

If your prototype has:

Smart animate on large frames

After-delay auto transitions

Heavy overlay stacks

Change them to:

Instant transitions

Mobile preview doesn’t need cinematic transitions.

Why This Is Happening

Your AVERRA aesthetic is:

Editorial

Image-heavy

Layered

Refined

That style often uses:

Large high-res imagery

Subtle shadows

Overlays

Wide whitespace

Beautiful.

But if built inefficiently, it becomes heavy.

You don’t need to change the design.
